## A React Hook to run media quries

Returns true or false depending on the provided media query.

### Component usage

In this example clicking the button calls `increment` which increments the `count` by 1

```jsx live
import { useMediaQuery } from './use-media-query';

const matches = useMediaQuery('(min-width: 240px) and (max-width: 767px)');
<h1>matches (min-width: 240px) and (max-width: 767px)?</h1>
<h2>{matches}</h2>
```
